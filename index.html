Projects

Fetal Health Classifier

A program used to classify the health of a fetus using ensemble learning technique to combine multiple traditional machine learning algorithms such as SVM, Decision Trees, Random Forest, KNN & Naive Bayes. The constituent outputs vote to select a resulting output for the overall system.

Stock Data Viewer

This webpage was built to demo my understanding of the D3 library. A Node.js server will feed you the closing stock price of META or TSLA from approximately Oct 2020 to approximately October 2021 and then your selection on the slider will decide the amount of datapoints shown. Each datapoint records the number of mouseover events (a guage of interest) and even displays the number of mouse over events.

Library Management System

Used QT to generate a User Interface for a Library Management System(LMS) in C++, it utilizes a variety of Priority Queues and Unordered Maps to enable the User to rapidly query the LMS for searches by Title and Author, while enabling sorts by Date, Size and Title. This program was developed with solid unit testing and a good understanding of OOP.

Calculator

A calculator I made in C++ which evaluates infix math expressions and evaluates them while respecting order of operations, this was done using Djikstra's Shunting Yard algorithm to generate a postfix expression and then solving it. If you enter expressions which are not supported such as scientific operations or unary operators, this program falls back on the Wolfram Alpha API to deliver the correct response. This program uses an MVC architecture and followed a Test Driven Development Methodology.

YouTube Group Personality Analysis Tool

This is a tool for researchers to generate data on Group or Amalgamated Personalities centered around Internet subcultures. I used the YouTube API to gather the top 100 comments under a video, and classify each comment along the four axis's of the MBTI personality scales in a Flask micro-framework.
